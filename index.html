LexTALE Test

English Test Instructions

This test consists of about 60 trials, in each of which you will see a string of letters. Your task is to decide whether this is a real English word or not. If you think it is an actual English word, click on "yes", and if you think it is not a real English word, click on "no".

If you are sure that the word exists, even though you don't know its exact meaning, you may still respond "yes". But if you are not sure if it is an existing word, you should respond "no". Be careful, however: Errors are penalised. So, there is no point in trying to increase your score by adding tallies to "words" you've never seen before!

The test uses British English rather than American English spelling—for example: realise instead of realize, and colour instead of color—but that does not really matter. The test is not about detecting such subtle spelling differences anyway.

You have as much time as you like for each decision. This test usually takes about 5 minutes.

If everything is clear, you can now start the experiment.

Instrucciones del Test en Español

Este test te va a mostrar 90 cadenas de letras, como relente y batinflar, de una a una.

En todos los casos, tu tarea consiste en decidir si se trata de una palabra del español, o si no lo es. Si crees que es una palabra de verdad, haz clic en «sí». Si crees que no existe en español, haz clic en «no». Así de simple.

Ojo: No hace falta conocer el significado exacto para responder «sí», pero si no tienes la certeza de que existe, es mejor responder «no». Una vez has escogido, no puedes volver atrás. Además, los errores se penalizan. Por tanto, no tiene sentido intentar aumentar la puntuación marcando palabras nunca vistas.

Tienes todo el tiempo que quieras para cada decisión, aunque es habitual tardar bastante menos de diez minutos. Si todo está claro, ya puedes empezar la prueba.
